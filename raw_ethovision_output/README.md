For the freezing, bursting, and cruising calculated by Ethovision
Open project file, and go to:
Analysis profiles
-Select Movement
-Select "movement", choose only "not moving" and change the stop velocity threshold to 5mm/s, and name that freezing.
- Select "velocity",  start at "5 mm/s", stop at "20 mm/s" and name that cruising.
- Select "velocity",  start at "20 mm/s", stop at the max mm/s in the experiment (maybe "1000 mm/s" ?) and name that bursting.

1 min time bin
